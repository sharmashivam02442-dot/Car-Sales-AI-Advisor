# 🚗 Car Sales AI Advisor

## 📌 Project Overview

Car Sales AI Advisor is a full-stack analytics application that combines data analytics, interactive dashboards, and AI-powered insights to help users analyze car sales performance and make data-driven decisions.

The application is built using Spring Boot, PostgreSQL, Streamlit, Plotly, and Ollama/Llama AI. It provides REST APIs for data processing, interactive visualizations for sales analysis, and an AI assistant capable of answering business-related questions using natural language.

---

## 🚀 Features

* 📈 Yearly Car Sales Analytics
* 📅 Monthly Sales Analysis
* 🤖 AI-Powered Business Insights
* 📊 Interactive Dashboard using Streamlit
* 📉 Sales Trend Visualization using Plotly
* 🔗 REST API Integration with Spring Boot
* 🗄️ PostgreSQL Database Connectivity
* ⚡ Real-Time Data Processing
* 🎯 User-Friendly Interface
